## TL;DR

**主论断**：离婚冷静期是 A7 强制基础设施在法律维度的典型形态——它以"保护家庭"的叙事遮蔽了其结构性功能：维持负外部性吸收结构，阻断女性退出 A4 提取闭环。

**边界条件句**：
1. 离婚冷静期分析不涉及 D0 的直接违反（D0 是再生产决策权，不是离婚决策权）
2. A7 失效峰值窗口命题在此适用——退出完成节点是暴力概率峰值
3. A7-Capture（施暴者攻击制度执行节点）在离婚冷静期中表现为法官自我审查

## 推导链骨架

```
P0：再生产循环不对称 → 系统产生结构性负外部性（男性暴戾/A4掠夺成本）
  ↓
制度需要稳定的负外部性吸收层（女性身体/劳动/情感作为缓冲）
  ↓
A7 的设计目标 = 维持缓冲结构稳定，不是保护个体退出
  ↓
离婚冷静期 = A7 法律维度的具体实现
  ↓
退出完成节点 = A4 提取链断裂信号 + A6 血统控制终止信号
  ↓
对施暴者：此刻是"最后机会"，暴力动机峰值
  ↓
对 A7 制度：退出完成后保护机制通常已解除，形成保护真空
  ↓
双重叠加：动机峰值 × 保护真空 = 暴力概率峰值
```

## 失效路径（3 步）

1. **失效条件 1**：若发现冷静期降低离婚后暴力发生率 → A7 失效峰值窗口命题失效
2. **失效条件 2**：若发现法官判决离婚率不受施暴者威胁影响 → A7-Capture 机制失效
3. **失效条件 3**：若发现立法史料中"保护个体退出权"优先于"维护家庭稳定" → A7 设计目标分析失效

## Fc 框架映射

### CCST 层面

```
离婚冷静期 = 恢复抽取结构在法律维度的制度实现

制度功能：
  表面：给双方"冷静时间"，减少冲动离婚
  实际：维持负外部性吸收结构，阻断女性退出 A4 提取闭环

恢复汇效应：
  A4 提取链断裂信号 → A6 血统控制终止信号 → 暴力动机峰值
  保护机制解除 → 保护真空
  双重叠加 → 暴力概率峰值
```

### A 系配置映射

| 配置 | 在离婚冷静期中的表现 |
|------|---------------------|
| **A4** 结构性掠夺 | 离婚 = A4 提取链断裂，施暴者试图恢复提取 |
| **A5** 外部承认 | "完整家庭"叙事压力，单亲污名 |
| **A6** 血统 ROI | 离婚 = A6 血统控制终止，施暴者试图恢复控制 |
| **A7** 强制基础设施 | 冷静期 = A7 法律维度的具体实现 |
| **A7-Capture** | 施暴者攻击制度执行节点（法官），导致自我审查 |

### A7 失效峰值窗口命题应用

```
退出完成节点（离婚/分手/调解完成瞬间及其后短时窗口）：
  ↓
A7 四维度失效的峰值时刻
  ↓
机制：
  退出完成 = A4 提取链断裂信号 + A6 血统控制终止信号
  → 对施暴者：此刻是"最后机会"，暴力动机峰值
  → 对 A7 制度：退出完成后保护机制通常已解除（调解结束/冷静期结束），形成保护真空
  → 双重叠加：动机峰值 × 保护真空 = 暴力概率峰值
  ↓
正条件：
  (1) 暴力发生时间与退出完成时间差 ≤ 72小时，或
  (2) 退出行为直接触发暴力升级（有时序证据）
  ↓
负条件：
  暴力发生与退出行为无时序关联，或先于退出行为
```

### A7-Capture 机制应用

```
施暴者通过攻击/威胁 A7 制度执行节点（法官/法庭/律师/调解员）：
  ↓
导致执行者产生自我审查
  ↓
A7 从"失效"变为"被征用为保护施暴者的机制"
  ↓
扩展阶段：
  阶段1：攻击女性本人（退出发起者）
  阶段2：攻击连带接触者（室友/家人/朋友）= A1 范围化
  阶段3：攻击退出的制度执行者（法官/调解员）= A7 被反向殖民
  ↓
执行者自我审查（"不敢判离"）→ A7 法律维度被征用为保护施暴者的机制
```

## 对抗测试（4 维）

### 结构对抗

**测试**：离婚冷静期分析是否过度依赖 A 系配置，忽略 G' 层物理约束？

**结果**：G' 层（物理/技术约束）在离婚冷静期中不直接适用。离婚冷静期是法律制度现象，不涉及身体物理成本。这是合法的适用域边界，不是分析缺失。

### 极端对抗

**测试**：若发现一个社会，其中离婚冷静期降低了离婚后暴力发生率，分析是否仍然成立？

**结果**：不成立。A7 失效峰值窗口命题会失效。这是有效的证伪条件。

### 反向对抗

**测试**：离婚冷静期是否可能是"保护双方"的结果？

**结果**：A7 设计目标分析已覆盖此反驳。"保护双方"在制度实际运行中被征用为"维持负外部性吸收结构"。但需补充：若发现冷静期内暴力发生率低于其他时段，则 A7 失效峰值窗口命题失效。

### 历史失败样本

**测试**：历史上是否有离婚冷静期改革的案例？

**结果**：
- **韩国"离婚熟虑期"**（2008年起）：有未成年子女的夫妻需经过3-6个月冷静期。实施后离婚率短暂下降，但家暴致死案未显著减少，证明冷静期不解决结构性暴力问题。
- **菲律宾**：无离婚合法化（仅"教会无效婚姻"），女性被锁定在婚姻中，是 A7 退出阻断的极端案例。
- **中国**：2021年民法典引入30天冷静期，实施后离婚率下降但家暴投诉率未显著变化。

## 一句话总结

离婚冷静期不是"保护家庭"或"减少冲动离婚"，是 A7 强制基础设施在法律维度的典型形态——它以"冷静"的叙事遮蔽了其结构性功能：维持负外部性吸收结构，阻断女性退出 A4 提取闭环，并在退出完成节点形成暴力概率峰值。
